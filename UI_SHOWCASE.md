## 📊 Design Metrics

### Spacing
- **Header padding**: 2rem (32px)
- **Section gaps**: 1.5-2rem (24-32px)
- **Card padding**: 1.8-2rem (28-32px)
- **Line height**: 1.5 (comfortable reading)

### Typography
```
H1 (Header): 2.5rem (40px) - Bold
H2 (Section): 1.8rem (28px) - Semi-bold
H3 (Cards):   1.2rem (19px) - Semi-bold
Body:         1rem (16px) - Normal
Small:        0.9rem (14px) - Normal
```

### Shadows
```
Light: 0 2px 8px rgba(0, 0, 0, 0.05)
Medium: 0 4px 12px rgba(0, 0, 0, 0.08)
Heavy: 0 8px 20px rgba(0, 0, 0, 0.12)
```

### Border Radius
```
Elements: 12px (modern, not too rounded)
Buttons/Tags: 8px (slightly less)
```
